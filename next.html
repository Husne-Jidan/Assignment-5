Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: it can include HTML tag and can add text inside an element
innerText: it can show just the text part of the element. so, we can use this to set the text to inside the element but we can't set the html tags whereas innerHTML can do.
textContent: it can set all text content inside the element also include the hidden text
